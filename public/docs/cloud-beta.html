# TAVP Cloud Beta > Managed hosting for TAVP Stack applications. ## What is TAVP Cloud? TAVP Cloud is a managed hosting platform for TAVP applications. It provides: - One-click deployment - Automatic scaling - SSL certificates - Backups - Monitoring ## Beta Features ### 1. Deployment ```bash # Deploy to TAVP Cloud tavp cloud:deploy # Check deployment status tavp cloud:status # View logs tavp cloud:logs ``` ### 2. Scaling - Auto-scaling based on traffic - Manual scaling via CLI or dashboard - CPU/Memory limits configurable ### 3. Database - Managed MySQL/PostgreSQL - Automatic backups - Point-in-time recovery ### 4. Storage - Managed file storage - S3-compatible API - CDN integration ### 5. Monitoring - Real-time metrics - Error tracking - Performance monitoring ## Pricing (Beta) | Plan | Price | Features | |------|-------|----------| | Starter | $19/mo | 1 app, 1GB RAM, 10GB storage | | Pro | $49/mo | 5 apps, 4GB RAM, 50GB storage | | Business | $149/mo | 20 apps, 16GB RAM, 200GB storage | | Enterprise | Custom | Unlimited apps, custom resources | ## Beta Access Apply for beta access at: https://tavp.dev/cloud/beta ## Status - **Beta Launch**: Q1 2026 - **Public Launch**: Q2 2026
