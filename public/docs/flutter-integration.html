# Flutter Integration Guide > Complete guide for integrating TAVP with Flutter mobile apps. ## Overview TAVP provides a RESTful API that Flutter apps can consume. This guide covers authentication, data fetching, and real-time features. ## Setup ### 1. TAVP API Configuration ```php // config/api.php return [ 'prefix' => 'api', 'middleware' => ['auth:api'], 'rate_limit' => 60, ]; ``` ### 2. Flutter Project Setup ```yaml # pubspec.yaml dependencies: http: ^1.1.0 flutter_secure_storage: ^9.0.0 web_socket_channel: ^2.4.0 ``` ## Authentication ### OTP Authentication (Email/SMS) ```dart class AuthService { final String baseUrl = 'https://your-api.com/api'; // Send OTP Future sendOtp(String email) async { final response = await http.post( Uri.parse('$baseUrl/auth/otp/send'), body: {'email': email}, ); return response.statusCode == 200; } // Verify OTP Future verifyOtp(String email, String code) async { final response = await http.post( Uri.parse('$baseUrl/auth/otp/verify'), body: {'email': email, 'code': code}, ); if (response.statusCode == 200) { final data = jsonDecode(response.body); return data['token']; // JWT token } return null; } } ``` ### JWT Token Storage ```dart class TokenStorage { final FlutterSecureStorage _storage = FlutterSecureStorage(); Future saveToken(String token) async { await _storage.write(key: 'auth_token', value: token); } Future getToken() async { return await _storage.read(key: 'auth_token'); } Future deleteToken() async { await _storage.delete(key: 'auth_token'); } } ``` ### API Client with Auth ```dart class ApiClient { final String baseUrl = 'https://your-api.com/api'; final TokenStorage _tokenStorage = TokenStorage(); Future> _headers() async { final token = await _tokenStorage.getToken(); return { 'Content-Type': 'application/json', 'Authorization': 'Bearer $token', }; } Future get(String endpoint) async { final response = await http.get( Uri.parse('$baseUrl$endpoint'), headers: await _headers(), ); return jsonDecode(response.body); } Future post(String endpoint, Map body) async { final response = await http.post( Uri.parse('$baseUrl$endpoint'), headers: await _headers(), body: jsonEncode(body), ); return jsonDecode(response.body); } } ``` ## Data Models ```dart class User { final int id; final String name; final String email; User({required this.id, required this.name, required this.email}); factory User.fromJson(Map json) { return User( id: json['id'], name: json['name'], email: json['email'], ); } } ``` ## Real-Time with WebSockets ```dart import 'package:web_socket_channel/web_socket_channel.dart'; class RealtimeService { WebSocketChannel? _channel; void connect(String token) { _channel = WebSocketChannel.connect( Uri.parse('wss://your-api.com/ws?token=$token'), ); _channel!.stream.listen((message) { final data = jsonDecode(message); // Handle real-time events }); } void subscribe(String channel) { _channel?.sink.add(jsonEncode({ 'action': 'subscribe', 'channel': channel, })); } void disconnect() { _channel?.sink.close(); } } ``` ## API Endpoints Reference ### Authentication | Method | Endpoint | Description | |--------|----------|-------------| | POST | `/api/auth/otp/send` | Send OTP | | POST | `/api/auth/otp/verify` | Verify OTP | | POST | `/api/auth/login` | Login with email/password | | POST | `/api/auth/logout` | Logout | ### Users | Method | Endpoint | Description | |--------|----------|-------------| | GET | `/api/user` | Get current user | | PUT | `/api/user` | Update user | | DELETE | `/api/user` | Delete user | ### Posts | Method | Endpoint | Description | |--------|----------|-------------| | GET | `/api/posts` | List posts | | GET | `/api/posts/:id` | Get post | | POST | `/api/posts` | Create post | | PUT | `/api/posts/:id` | Update post | | DELETE | `/api/posts/:id` | Delete post | ## Error Handling ```dart class ApiException implements Exception { final int statusCode; final String message; ApiException({required this.statusCode, required this.message}); @override String toString() => 'ApiException: $statusCode - $message'; } class ApiClient { // ... Future get(String endpoint) async { final response = await http.get( Uri.parse('$baseUrl$endpoint'), headers: await _headers(), ); if (response.statusCode >= 200 && response.statusCode < 300) { return jsonDecode(response.body); } throw ApiException( statusCode: response.statusCode, message: jsonDecode(response.body)['message'] ?? 'Unknown error', ); } } ``` ## Best Practices 1. **Token Security** — Store tokens in secure storage, never in plain text 2. **Error Handling** — Always handle API errors gracefully 3. **Loading States** — Show loading indicators during API calls 4. **Caching** — Cache responses to reduce API calls 5. **Offline Support** — Implement offline-first architecture ## Example App ```dart void main() { runApp(MyApp()); } class MyApp extends StatelessWidget { @override Widget build(BuildContext context) { return MaterialApp( title: 'TAVP Flutter', theme: ThemeData( primarySwatch: Colors.blue, visualDensity: VisualDensity.adaptivePlatformDensity, ), home: HomeScreen(), ); } } class HomeScreen extends StatefulWidget { @override _HomeScreenState createState() => _HomeScreenState(); } class _HomeScreenState extends State { final ApiClient _api = ApiClient(); List _posts = []; bool _loading = true; @override void initState() { super.initState(); _loadPosts(); } Future _loadPosts() async { try { final data = await _api.get('/posts'); setState(() { _posts = (data['data'] as List) .map((json) => Post.fromJson(json)) .toList(); _loading = false; }); } catch (e) { setState(() => _loading = false); ScaffoldMessenger.of(context).showSnackBar( SnackBar(content: Text('Failed to load posts')), ); } } @override Widget build(BuildContext context) { return Scaffold( appBar: AppBar(title: Text('TAVP Posts')), body: _loading ? Center(child: CircularProgressIndicator()) : ListView.builder( itemCount: _posts.length, itemBuilder: (context, index) { final post = _posts[index]; return ListTile( title: Text(post.title), subtitle: Text(post.excerpt), ); }, ), ); } } ``` ## Need Help? - [TAVP Discord](https://discord.gg/tavp) - [Flutter Documentation](https://flutter.dev/docs)
