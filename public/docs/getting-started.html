# Getting Started > Quick start guide for TAVP Stack. ## Requirements - PHP 8.3+ - Phalcon 5.16+ - Node.js 18+ - Composer 2.x ## Installation ```bash # Install TAVP CLI composer global require tavp/cli # Create new project tavp new my-app # Or use composer directly composer create-project tavp/core my-app # Enter directory cd my-app # Install dependencies composer install npm install # Start development server tavp serve ``` ## Project Structure ``` my-app/ ├── app/ │ ├── Controllers/ │ ├── Models/ │ ├── Middleware/ │ └── Services/ ├── config/ ├── database/ │ ├── migrations/ │ └── seeders/ ├── public/ │ └── index.php ├── resources/ │ └── views/ ├── routes/ ├── storage/ ├── tests/ └── vendor/ ``` ## First Steps ### 1. Configure Environment ```bash cp .env.example .env tavp key:generate ``` ### 2. Configure Database ```bash # Edit .env DB_CONNECTION=mysql DB_HOST=127.0.0.1 DB_DATABASE=tavp DB_USERNAME=root DB_PASSWORD=secret ``` ### 3. Run Migrations ```bash tavp migrate ``` ### 4. Create First Model ```bash tavp make:model Post ``` ### 5. Create Controller ```bash tavp make:controller PostController ``` ### 6. Add Route ```php // routes/web.php $router->get('/', function () { return view('home'); }); $router->get('/posts', [PostController::class, 'index']); ``` ### 7. Create View ```php {% extends 'layouts/app.volt' %} {% block content %}

Welcome to TAVP

{% endblock %} ``` ### 8. Start Development ```bash tavp serve ``` Open http://localhost:8000 ## Next Steps - [Architecture Overview](/docs/architecture) - [Database Guide](/docs/database) - [ORM Guide](/docs/orm) - [Authentication](/docs/authentication) - [Deployment](/docs/deployment)
